Principios de Aplicación

Reglas para mantener consistencia en todo el ecosistema

▤

Sistema de Grid y Espaciado

Sistema de grid de 8 puntos (8pt) para consistencia visual: espaciado entre párrafos: 16px (2 × 8px), padding interno de tarjetas: 24px (3 × 8px).

Aa

Jerarquía Tipográfica

Usa Libre Baskerville para títulos y Inter para texto normal. Mantén una línea base clara y legible para mejorar la experiencia de lectura.

🎨

Paleta de Colores

Utiliza la paleta específica de Anclora Press: #23436B, #FFC979, #FFECB3, #F6F7F9, #162032 para mantener coherencia visual en toda la aplicación.
